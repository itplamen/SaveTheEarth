You start the game with 3 lives and 0 points. The purpose of the game is to stay alive and destroy as many asteroids as you can. There are different bonuses and asteroids. Their positions are random.

Asteroids

The asteroids are with random size and speed. You can gain points from every asteroid destruction. For every destroyed asteroid you gain some points, which value depends from asteroid's size and speed. The number of the asteroids depends from the current game level. The required number of hits for asteroids destruction depends from the asteroid size.

Bonuses

You can also gain points if you get 'bonus points', which gives you 100 points. The number of the bonuses depends from the current game level. There is no strict rule for which bonus should be displayed. The game may display two 'live bonuses' or one 'speed bonus' and one 'live bonus'. The types of the bonuses are listed below:

  1. Live - gives you +1 live.
  2. Points - gives you +100 points.
  3. Rockets - gives you +2 rockets.
  4. Shield - gives you +1 shield.
  5. Speed - gives you +1 speed.

Spaceship

The spaceship has unlimited number of lasers. Each time when the 'Space' key is pressed (or the key for 'Fire Lasers' action if you changed the default 'Space' key), the spaceship fires two lasers. Your number of lives decrease if you crash into asteroid. When the number of lives is 0 - Game Over! If you have missiles, you can fire them at every time you want. The first missile is fired from the left wing, the second missile - from the right wing. If you have shield, it can be activated at every time. There are 6 types of spaceships. Every spaceship has special ability - nuclear bomb or squadron. You can use the special ability only 3 times during the game. Every spaceship has different speed, strength and damage.

Game

The game gives you the opportunity to save and load your game. You can save as many games as you want. When the game ends, you can save your score (points) into the ranking list.

Menu

You can see the default keys and actions in 'Options' section. You can also change some of the keys and after that you can restore the default keys.
